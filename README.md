TiU Turportal 2025 Gruppe 2

Utvikling av en komplett webløsning for turinteresserte med integrert annonseportal og fellesturfasiliteter. Del av USN-prosjektet 2025/2026.

For å teste vær funksjonen må man åpne en ny terminal som man skriver "node weather-proxy.js" for å opprette en proxy server. Denne må kjøre en egen terminal og man kan ikke starte nettsiden fra samme terminal.
PS. Hvis dette er første gang du tester dette så man skrive "npm install cors".

For å teste nettsiden så må du være inni i "tiu-app". Bruk "cd .." for å gå tilbake og bruk cd "mappe_navn" for å gå dypere i mappene.
Deretter kan du skrive "npm start" i terminalen.
